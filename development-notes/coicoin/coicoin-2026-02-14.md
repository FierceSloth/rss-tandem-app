## Решения:

- После обсуждения на код-ревью было решено отказаться от локальных SCSS-констант в пользу CSS-переменных, чтобы сделать их глобально доступными. Это решает проблему повторного объявления переменных в каждом файле и упрощает поддержку стилей.
- Также обсудили вариант с добавлением preprocessorOptions в vite.config.ts, но в итоге остановились на варианте с CSS.

```js
css: {
    preprocessorOptions: {
      scss: {
        additionalData: `
          @use "@/styles/constants/constants" as *;
        `,
      },
    },
  },
```

## Проблемы:

- При попытке совместить SCSS-миксины и CSS-переменные выявилась техническая проблема, когда SCSS не может вычислить var(--font-size-large) как аргумент миксина при компиляции.
- В итоге пришли к решению удалить mixin font, оставить mixins для адаптации экранов desktop и телефона.

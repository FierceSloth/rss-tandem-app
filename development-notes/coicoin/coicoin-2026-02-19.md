## Решения:

- Для навигации по страницам решил использовать аналог useNavigate() из ReactJS, под капотом используется Emitter.
- Для получения параметров URL решил добавить хук useParams().

Пример:

```ts
export function useParams(): Record<string, string> {
  return router.getParams();
}

export function useNavigate({ page, parameters = {}, query = {} }: IPathData): void {
  appEmitter.emit(ROUTER_NAVIGATE, { page, parameters, query });
}
```

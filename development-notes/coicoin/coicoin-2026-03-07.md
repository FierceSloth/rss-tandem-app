## Проблемы

- Обнаружены лишние амперсанды (`&`) в SCSS.
- Возникла проблема с адаптацией: изначально использовался `grid`, и на мобильных устройствах всё работало корректно.
  После перехода на `flex` code editor перестал корректно сжиматься на мобильных устройствах, из-за чего отсутствовал горизонтальный скролл.

## Решения

- Удалил лишние `&` в SCSS по рекомендации коллеги, так как в ряде мест они были избыточными.
- Рассматривал вариант вернуть `grid`, однако в итоге было найдено решение:

  ```
  :global(.cm-content) {
    width: 1px;
  }
  ```

  Это позволило корректно работать горизонтальному скроллу в code editor.

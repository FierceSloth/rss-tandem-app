10.02.2026 Первый командный созвон

На созвоне наша команда обсудила стиль и концепцию приложения.
Решено сделать проект как Widget Trainer, с выбором темы (модуля) и с 3-7 уровнями внутри модуля,
которые будут реализованы как мини-игры (виджеты).

Мини-игры будет делиться на два типа
Легкие: Quiz Round, True/False на время, Output Predictor и др.
Сложные: Solving Tasks with a code editor, Error corrector и др.

В конце каждого модуля будет финальный уровень с повышенной сложностью

При реализации проекта по началу будут главные и необходимые проекту компоненты и легкие мини-игры,
а затем при свободном времени, реализовыватся остальная часть проекта

User Flow и дизайн схема описана в [Figma](https://www.figma.com/design/g6Zb6zh7WaxWGElPEHlSy8/Tandem?node-id=0-1&p=f&t=Mt7SXonjmtwQ4HhD-0)
Если в кратце то: Landing Page → Регистрация/Вход → Dashboard → Выбор темы(модуля) → Выбор уровня → Практика(Виджет) → Финальный уровень → Завершение темы(модуля)

Дизайн будет реализовываться по такой схеме:
Промпт → AI верстка (variant.com) → Импорт HTML в Figma → Ручная корректировка → Финальный макет

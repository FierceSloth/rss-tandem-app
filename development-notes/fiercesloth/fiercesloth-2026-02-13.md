## Решения:

- Во время ревью, я предложил использовать css переменные, вместо scss переменных. Так как к css переменным можно обратится в любом куске коде, без лишнего импорта, а так же убирают проблему двойных scss перменных для desktop и mobile версии, потому что их можно переопределить в медиа запросе. И мы приняли решение сделать все на css перменных.
